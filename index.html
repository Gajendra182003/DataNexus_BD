📊 Project Overview

DataNexus BD implements a production-grade Bronze → Silver → Gold Medallion Architecture ETL pipeline on the Databricks Lakehouse platform for multi-domain analytics.

The project combines semiconductor stock data, ESG risk ratings, and renewable energy data to create analytics-ready Gold tables and business dashboards.

🚀 Quick Start

Prerequisites

  • Databricks Workspace
  • Unity Catalog enabled
  • Serverless Compute or Databricks Runtime 13.x+
  • Python 3.9+ if regenerating data locally

Installation

git clone https://github.com/Gajendra182003/DataNexus_BD.git
cd DataNexus_BD

Run the Pipeline

  1. Upload the project folder to Databricks Workspace.
  2. Open DataNexus_Pipeline.py or import it as a Databricks notebook.
  3. Attach Serverless Compute or a suitable cluster.
  4. Run the cells in order: Bronze → Silver → Gold → Dashboard queries.
  5. Open the Databricks SQL dashboard named DataNexus BD Analytics.
